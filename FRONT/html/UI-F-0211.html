  • 회원가입
  • 로그인

기존에 가입하신 경우이기 때문에 비밀번호 찾기를 통해 계정 정보 확인 후 로그인 진행해주시면 됩니다.

사업자 번호로 로그인 하기 때문에 한 기업에서 하나의 계정만 생성 가능합니다. 단, 본사/지사가 따로 있는 경우 각각 계정 생성이 가능합니다.

본사/지사별로 회원가입을 진행해주시면 됩니다. 회원가입 시 사업자등록번호 인증 단계에서 지사의 사업자등록번호를 이용하여 회원가입합니다. 등록되지 않았을 경우 신규등록으로 회원가입이 가능합니다.

비밀번호 찾기가 어려울 경우, 안내되어있는 이메일(abcd_@gntp.com(미정))을 통해 문의해주시기 바랍니다. 이메일 문의의 경우 처리 완료까지 최대 2~3일 소요될 수 있습니다.

비밀번호 입력 오류 5회 이상일 경우 자동 잠금 처리됩니다.

[마이페이지 > 기본정보] 우측 기업명 하단에 있는 ‘비밀번호 변경’ 버튼을 클릭하여 변경하시면 됩니다.
